# DONGLE 관리자 앱 기능 소개

## 문서 목적
DONGLE 관리자 앱에서 어떤 일을 할 수 있는지 쉽게 설명한 안내서입니다.
처음 사용하는 분도 이 문서만 읽으면 화면별 역할을 이해할 수 있습니다.

## 서비스 접속 주소
admin.dev.dongle.wavers.kr

## 누가 사용하는 서비스인가요
### 총동연 관리자
- 전체 동아리와 사용자, 배너, 활동보고서를 관리합니다.

### 동아리 담당자
- 본인 동아리의 정보와 활동보고서를 관리합니다.

### 신규 등록 사용자
- 관리자에게 받은 등록 링크로 동아리 등록을 진행합니다.

## 총동연 관리자가 할 수 있는 일
### 홈 화면
- 주요 관리 메뉴로 빠르게 이동할 수 있습니다.
- 동아리 관리, 동아리 등록, 활동보고서 관리, 사용자 관리, 배너 관리로 이동합니다.

### 동아리 정보 관리
- 전체 동아리 목록을 확인합니다.
- 각 동아리 정보를 열어 수정할 수 있습니다.
- 회장 이름과 연락처를 수정할 수 있습니다.
- 더 이상 필요 없는 동아리는 삭제할 수 있습니다.

### 동아리 등록 링크 발급
- 신규 동아리 등록용 링크를 만들 수 있습니다.
- 만든 링크를 복사해서 전달할 수 있습니다.
- 화면에 안내되는 주의사항을 통해 등록 진행 방법을 확인할 수 있습니다.

### 활동보고서 관리
- 동아리별 활동보고서 목록을 볼 수 있습니다.
- 각 보고서의 상세 내용을 확인할 수 있습니다.
- 제목, 작성일, 내용, 첨부 이미지를 확인할 수 있습니다.

### 사용자 관리
- 등록된 사용자 목록을 볼 수 있습니다.
- 새 사용자를 추가할 수 있습니다.
- 기존 사용자 정보를 수정할 수 있습니다.
- 더 이상 필요 없는 사용자를 삭제할 수 있습니다.

### 메인 배너 관리
- 현재 배너 목록을 볼 수 있습니다.
- 새 배너를 등록할 수 있습니다.
- 기존 배너의 이미지, 노출 기간, 사용 여부를 수정할 수 있습니다.
- 배너를 삭제할 수 있습니다.

## 동아리 담당자가 할 수 있는 일
### 동아리 정보 관리
- 동아리 이름, 소개, 활동 정보, 연락처 정보를 관리합니다.
- 회장 정보도 함께 관리할 수 있습니다.

### 활동보고서 관리
- 활동보고서를 새로 작성할 수 있습니다.
- 작성한 보고서를 상세 화면에서 확인할 수 있습니다.
- 보고서를 수정할 수 있습니다.
- 보고서를 삭제할 수 있습니다.

### 계정 정보 관리
- 본인 아이디를 변경할 수 있습니다.
- 본인 비밀번호를 변경할 수 있습니다.

## 신규 동아리 등록 화면에서 할 수 있는 일
### 등록 정보 입력
- 동아리 기본 정보를 입력해 신규 등록을 완료할 수 있습니다.

### 등록 완료 확인
- 등록이 끝나면 완료 화면이 보입니다.
- 임시 로그인 정보를 확인하고 복사할 수 있습니다.
- 다음에 해야 할 작업 안내를 확인할 수 있습니다.

## 사용 중 보게 되는 안내 방식
### 입력 안내
- 필수 입력값이 비어 있으면 입력이 필요하다는 안내가 표시됩니다.
- 형식이 맞지 않으면 수정이 필요하다는 안내가 표시됩니다.

### 저장 결과 안내
- 정상 처리되면 성공 안내가 표시됩니다.
- 문제가 있으면 오류 안내가 표시됩니다.

### 삭제 전 확인
- 삭제 버튼을 누르면 다시 묻는 확인 창이 표시됩니다.
- 실수로 삭제하지 않도록 보호 장치가 있습니다.

## 이 서비스가 특히 유용한 상황
- 학기 초 신규 동아리 등록이 많을 때
- 동아리 정보를 최신 상태로 유지해야 할 때
- 활동보고서를 한 곳에서 확인해야 할 때
- 운영 계정과 권한을 정리해야 할 때
- 메인 화면 홍보 배너를 주기적으로 교체해야 할 때
