# Product

이 디렉터리는 동글 제품 기획을 최소 문서로 정리한다.

## 문서 구성

- [features.md](features.md)
    - 기능별 요구사항, 화면 흐름, 주요 정책
- [decisions.md](decisions.md)
    - 확정된 기획 결정, 보류 항목, 변경 이유

## 서비스 개요

동글은 동아리 정보를 사용자에게 제공하고, 관리자와 회장이 동아리 운영 정보를 관리하는 서비스다.

## 사용자 역할

| 역할        | 설명                                      | 주요 기능                                              |
| ----------- | ----------------------------------------- | ------------------------------------------------------ |
| 일반 사용자 | 로그인 없이 동아리 정보를 탐색하는 사용자 | 동아리 목록, 동아리 상세, 활동보고서 조회              |
| 회장        | 담당 동아리 운영 정보를 관리하는 사용자   | 동아리 정보 수정, 계정 정보 관리, 활동보고서 작성/수정 |
| 관리자      | 서비스 운영을 관리하는 사용자             | 동아리 관리, 등록 URL 발급, 사용자 관리, 배너 관리     |

## 앱 구성

- `DONGLE-CLIENT`: 일반 사용자용 앱
- `DONGLE-ADMIN`: 관리자와 회장용 앱
